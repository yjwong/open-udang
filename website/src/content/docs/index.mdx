## Features

<CardGrid stagger>
	<Card title="Multi-project contexts" icon="open-book">
		Switch between projects with `/context`. Each context has its own working directory, model, allowed tools, and session history.
	</Card>
	<Card title="Tool approval" icon="approve-check-circle">
		Every tool call goes through path-scoped approval. Read-only tools auto-approve within your project; mutating tools always ask first. You stay in control.
	</Card>
	<Card title="Sandboxed execution" icon="seti:docker">
		Run the agent in a Docker container, a Libvirt VM, or a macOS sandbox. Filesystem isolation means you can safely auto-approve everything.
	</Card>
	<Card title="Computer use" icon="laptop">
		Enable a headless desktop inside the sandbox. The agent can take screenshots, click, type, and browse the web — viewable live via VNC.
	</Card>
	<Card title="Scheduled tasks" icon="seti:clock">
		Set up cron-like recurring prompts in natural language. The agent runs them automatically and reports back to your chat.
	</Card>
	<Card title="Mini Apps" icon="puzzle">
		Review diffs, view terminal output, watch VNC, and preview markdown — all through Telegram Mini Apps built into the bot.
	</Card>
</CardGrid>
